# Student Profile

## Purpose

Represents the immutable or slowly changing identity of a learner.

## Responsibilities

- Store demographic information.
- Store academic identity.
- Provide a stable identifier for the Digital Twin.

## Out of Scope

- Skills
- Knowledge
- Goals
- Preferences
- Memories
- Reflections

## Research Rationale

Separating learner identity from evolving educational attributes
reduces coupling and enables independent experimentation on
learning models without modifying identity data.
